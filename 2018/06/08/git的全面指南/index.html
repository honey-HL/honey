git的全面指南
git使用指南
  1. 先用git init初始化git
  2. git branch honney创建新的分支
  3. git checkout master切换到master分支
  4. 用git status查看当前分支状态
  5. 用 git add . 将修改的文件提交暂存库
  6. 用git commit - m honney(本地分支名)将已经修改的文件提交到本地分支
  7. git push origin honney将本地分支上传，仓库会多一个你的分支，期间要求输入gitHub的username 和 password

  8. git remote add origin + 远程仓库地址，连接远程仓库

  9. git push --set-upstream origin honney设置要追踪的远程分支（即自己要push的远程分支）

  10. git push把本地仓库的所有代码上传到自己分支所在的远程仓库

  11. git branch -vv 查看本地分支追踪的远程分支以及提交情况(ahead或者behind)
